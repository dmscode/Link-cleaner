这是什么：
---

这是一个小书签（Bookmarklet），把 js 文件的内容当做网址保存为书签，在需要的时候点击一下这个书签，就可以将当前网页的网址净化（去除不必要的 get 参数）。

默认情况会自动将净化后的网址保存当剪切板，并将当前页面转向至净化后的网址，以便验证可用性。当写入剪切板失败的时候会弹出提示框，可以自行复制网址，确认后网页自动跳转进行验证。
